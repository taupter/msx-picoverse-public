# PicoVerse 2350 Case Files

This folder contains the 3D model files for the PicoVerse 2350 cartridge case.

## Files

- `Cartouche_MSX_V2_FLAT_PICOVERSE_2350_1.0_A.obj` - case shell part A for the PicoVerse 2350 case version 1.0.
- `Cartouche_MSX_V2_FLAT_PICOVERSE_2350_1.0_B.obj` - case shell part B for the PicoVerse 2350 case version 1.0.
- `PicoVerse_2350_button.stl` - printable button part for the PicoVerse 2350 1.1/1.2 case.
- `PicoVerse_2350_1.1_Paco_boton_4mm.stl` - printable button part for the PicoVerse 2350 1.1 case.
- `PicoVerse_2350_1.1_Paco_bottom.stl` - printable bottom part for the PicoVerse 2350 1.1 case.
- `PicoVerse_2350_1.1_Paco_top.stl` - printable top part for the PicoVerse 2350 1.1 case.
