# PicoVerse 2040 Case Files

This folder contains 3D model and drawing files for the PicoVerse 2040 cartridge case.

## Files

- `PicoVerse_2040_1.41_bootsel_hole.svg` - SVG drawing for the PicoVerse 2040 version 1.41 BOOTSEL access hole.
- `PicoVerse_2040_1.41_button.stl` - printable button part for the PicoVerse 2040 case version 1.41.
- `PicoVerse_2040_1.41_label_holder.stl` - printable label holder part for the PicoVerse 2040 case version 1.41.
- `PicoVerse_2040_button_pakoto.stl` - printable button part from the Pakoto case set.
- `PicoVerse_2040_side1_pakoto.stl` - printable case shell side 1 from the Pakoto case set.
- `PicoVerse_2040_side2_pakoto.stl` - printable case shell side 2 from the Pakoto case set.
